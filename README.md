## LOCARS
Logiciel Optofluidique de Contrôle pour l’Acquisition par Raman Spectroscopy (LOCARS).
Ce logiciel a été développé dans le cadre de la maîtrise de Tommy Brasseur à Polytechnique Montréal. Il permet le contrôle d’un scanner Raman automatisé pour l’acquisition de spectres sur des explants tumoraux intégrés dans des plateformes microfluidiques.

## Installation
Installer les dépendances Python :
pip install -r requirements.txt

## Lancer le logiciel
Depuis le dossier du projet :
python LOCARS.py
Contexte
Ce logiciel fait partie de la plateforme optofluidique développée pour l’acquisition longitudinale de spectres Raman sur des explants tumoraux viables.
